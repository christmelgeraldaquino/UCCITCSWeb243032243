Introduction to VS Code

Visual Studio Code (VS Code) is a free, open-source code editor developed by Microsoft. It is one of the most popular code editors used by developers worldwide due to its lightweight design, extensive customization options, and powerful features.

VS Code supports a wide range of programming languages and comes with built-in support for JavaScript, TypeScript, HTML, CSS, and many more. It features IntelliSense for code completion, debugging capabilities, Git integration, and a vast ecosystem of extensions.

One of the key advantages of VS Code is its keyboard shortcuts system, which allows developers to work more efficiently without constantly switching between mouse and keyboard. Mastering these shortcuts can significantly improve your coding productivity.

VS Code is available for Windows, macOS, and Linux, making it accessible to developers on any platform. It's built on Electron and uses the same editor core as Azure DevOps.

To get started with VS Code, you can download it from the official website. Once installed, you can customize it with themes, extensions, and keyboard shortcuts to match your workflow preferences.

// Example: Opening VS Code from terminal code . // Opens current directory code filename.js // Opens specific file code -r . // Reopens current window

VS Code's Command Palette (accessed with Ctrl+Shift+P on Windows/Linux or Cmd+Shift+P on Mac) is a powerful feature that allows you to access almost any command or setting in the editor.

VS Code Shortcuts

Mastering VS Code keyboard shortcuts can significantly improve your coding productivity. Here are the most essential shortcuts organized by category.

File and Editor Navigation:

  • Ctrl+P (Windows/Linux) or Cmd+P (Mac) - Quick Open: Quickly open files by name
  • Ctrl+Shift+P (Windows/Linux) or Cmd+Shift+P (Mac) - Command Palette: Access all commands
  • Ctrl+Tab - Switch between open editors
  • Ctrl+\ - Split editor
  • Ctrl+1, Ctrl+2, Ctrl+3 - Focus on editor groups 1, 2, or 3

Code Navigation:

Ctrl+G // Go to Line Ctrl+Shift+O // Go to Symbol in File Ctrl+T // Go to Symbol in Workspace F12 // Go to Definition Alt+F12 // Peek Definition Shift+F12 // Go to References Ctrl+Shift+F // Search in Files Ctrl+F // Find in File Ctrl+H // Replace in File

Cursor Movement:

  • Ctrl+Left/Right Arrow - Move cursor by word
  • Home/End - Move to beginning/end of line
  • Ctrl+Home/End - Move to top/bottom of file
  • Alt+Up/Down Arrow - Move line up/down

Text Selection and Editing:

Ctrl+D // Select word at cursor (repeat to select next occurrence) Ctrl+Shift+L // Select all occurrences of current selection Ctrl+L // Select entire line Ctrl+Shift+K // Delete line Ctrl+Enter // Insert line below Ctrl+Shift+Enter // Insert line above Ctrl+] // Indent line Ctrl+[ // Outdent line

Multi-Cursor Editing:

  • Alt+Click - Add cursor at click position
  • Ctrl+Alt+Up/Down - Add cursor above/below
  • Ctrl+Shift+Alt+Up/Down - Add cursor to all occurrences
  • Ctrl+U - Undo last cursor operation

File Operations:

Ctrl+N // New File Ctrl+O // Open File Ctrl+S // Save Ctrl+Shift+S // Save As Ctrl+K S // Save All Ctrl+W // Close Editor

Explorer and Sidebar:

  • Ctrl+Shift+E - Focus on Explorer
  • Ctrl+B - Toggle sidebar visibility
  • Ctrl+` - Toggle terminal
  • Ctrl+J - Toggle panel (terminal, problems, etc.)
  • Ctrl+, - Open Settings
  • Ctrl+K Ctrl+S - Open Keyboard Shortcuts

These shortcuts allow you to work efficiently without constantly switching between mouse and keyboard, significantly improving your coding speed and workflow.

Introduction to Emmet

Emmet is a web developer's toolkit that can greatly improve your HTML and CSS workflow. It's built into VS Code by default and allows you to write HTML and CSS using a shorthand syntax that expands into full code.

Emmet uses abbreviations that look like CSS selectors, which you can expand into full HTML or CSS code. This dramatically speeds up your coding process, especially when creating repetitive structures.

To use Emmet in VS Code, simply type an abbreviation and press Tab or Ctrl+E (Windows/Linux) or Cmd+E (Mac) to expand it. Emmet works in HTML, CSS, JavaScript, JSX, and other markup languages.

Here's a simple example: Type div.container and press Tab, and it expands to:

<div class="container"></div>

Emmet understands HTML structure, CSS properties, and can generate complex nested structures with just a few keystrokes. It's particularly powerful when creating layouts, forms, and lists.

Key Emmet Features:

  • Element Generation: Create HTML elements quickly
  • Nesting: Use > for child elements and + for siblings
  • Attributes: Add classes, IDs, and other attributes easily
  • Multiplication: Use * to repeat elements
  • Numbering: Use $ for automatic numbering
  • Text: Add text content with {}

Emmet is one of the most powerful productivity tools for web developers, and learning it can save you hours of typing repetitive code.

Common Emmet Abbreviations

Here are some of the most commonly used Emmet abbreviations that will help you write HTML and CSS faster.

Basic HTML Structure:

! // HTML5 boilerplate div // <div></div> div.container // <div class="container"></div> div#header // <div id="header"></div> div.container#main // <div class="container" id="main"></div>

Nesting and Siblings:

div>p // <div><p></p></div> div>p>span // Nested: div > p > span div+p // Siblings: div and p side by side div.container>ul>li // Container with list

Multiplication and Numbering:

ul>li*5 // 5 list items ul>li.item$*5 // Numbered: item1, item2, item3... div.box$*3 // 3 divs with classes box1, box2, box3

Attributes:

  • a[href="#"] - Link with href attribute
  • img[src="image.jpg"][alt="description"] - Image with multiple attributes
  • input[type="text"][placeholder="Enter text"] - Input with attributes
  • div[data-id="123"] - Custom data attributes

Text Content:

p{Hello World} // <p>Hello World</p> a{Click here} // <a href="#">Click here</a> h1{Main Title}+p{Description}

Complex Examples:

nav>ul>li*5>a[href="#"]{Link $} // Creates navigation with 5 links form>input[type="text"][name="username"]+input[type="password"][name="password"]+button{Submit} // Creates a login form div.container>header#header+main#main>section*3>h2{Section $}+p{Lorem ipsum}+footer#footer // Creates a complete page structure

CSS Abbreviations:

  • m10 - margin: 10px;
  • p20 - padding: 20px;
  • w100 - width: 100px;
  • h50 - height: 50px;
  • bgc#fff - background-color: #fff;
  • d:f - display: flex;

Practice these Emmet abbreviations regularly, and you'll find yourself writing HTML and CSS much faster. Start with simple abbreviations and gradually work your way up to more complex structures.
